# College Companion UI

This project implements the UI/UX for the College Companion app.

## Features

- **Modern Theme**: Custom color palette (Royal Blue & Emerald Green) with clean typography.
- **Dashboard**: Summary card and Recent Activity list using RecyclerView.
- **Voice Interface**: Persistent voice interaction area with pulse animation.
- **Navigation**: Bottom Navigation Bar switching between Home, Timetable, and Reminders.
- **Settings**: Dedicated settings screen with toggle switches.

## Structure

- `MainActivity`: Hosts the Navigation component and Voice UI.
- `HomeFragment`: Displays the main dashboard.
- `TimetableFragment` / `RemindersFragment`: Placeholder screens for future implementation.
- `SettingsActivity`: Configuration screen.

## Icons

Vector drawables are used for all icons to ensure crisp rendering on all devices.
